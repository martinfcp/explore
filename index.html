SEREMI de Salud Los Ríos

A mi llegada al departamento de Epidemiología, las falencias en cuanto a manejo de datos eran claras, problemas como la inexistencia de una base de datos como
tal, el nulo flujo de datos entre secciones del departamento o la falta de identidad y complejidad en los reportes serían temas que deberían ser abordados para
dar un salto de calidad y mejorar tanto la posibilidad de analizar los datos que generaba el área de trazabilidad, así como la entrega de información interna y
externa. Algunos de estos proyectos serán abordados, aclarando que, ninguna información de carácter interno será filtrada, aquí solo se podrán encontrar datos
de distribución pública disponibles en la página del MINSAL y datos ficticios para hacer una demostración de las funciones de las herramientas




Herramienta de streaming de datos Epidemiología SEREMI de Salud Los Ríos

Esta herramienta se creó para cubrir necesidades de varios actores del área, la sección de Brotes necesitaba una manera de detectar primeras alertas de brotes
que se pudieran dar en distintas situaciones, empresariales, por reuniones sociales, convivencia institucional, etc. Desde trazabilidad necesitaban además un
que uniera las investigaciones de todos los trazadores en una para tener una mejor visión del panorama y así poder encontrar correlaciones más fácilmente, por
otra parte, desde el área de gestión de la información se necesitaba una manera de homologar datos para hacer la limpieza de estos mismos de manera más eficiente
todo esto debía ser con la mayor simpleza posible para que el usuario más principiante de hojas de cálculo pudiese usarlo. De todo esto nace esta herramienta que permitiría
realizar todas estas funciones, nutrida de una base de datos de empresas recopiladas por el área de investigación laboral para que se pudiesen encontrar coincidencias
y se pudiese llevar un registro del comportamiento de estas con respecto a la contingencia de pandemia.
